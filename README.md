## 프로젝트 설명
매년 12월, 새하얀 눈이 내리는 거리에 캐롤이 울려퍼지면 설레하며 크리스마스를 기다리는 우리의 모습을 볼 수 있습니다. 바쁜 일상은 잊고 잠시 어린 아이로 돌아가 귀여운 캐릭터들이 등장하는 게임과 함께 크리스마스를 기다려보는 것은 어떨까요?

**크리스마스 선물을 되찾아라!** Pygame을 이용한 러닝 액션 게임 '**Santalim-With-Gildolph**'입니다.
